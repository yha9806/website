## Production Notes

- **Critical**: The stopwatch must run continuously and visibly in the corner — the time pressure is the narrative device
- **Screen Recording**: Use a clean desktop, large font, dark terminal theme
- **No cuts in the screen recording** — this must feel like one continuous, real-time take (speed up install step if needed, but show it)
- **Aspect Ratio**: 16:9 primary (demo-style), also cut 9:16 for Shorts/Reels
- **Captions**: Always on, EN
- **Thumbnail**: Stopwatch showing "60s" next to VULCA interface screenshot
