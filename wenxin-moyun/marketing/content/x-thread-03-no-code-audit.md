## Tweet 2 — Step 1: Install

Step 1: Install VULCA

```
pip install vulca
```

That's it. One package. All dependencies handled. Works on Python 3.9+.

Time elapsed: 15 seconds.

#DevTools

> 第一步：安装。pip install vulca。就这样，一个包，所有依赖自动处理，Python 3.9+。已用时：15秒。

---

## Tweet 3 — Step 2: Launch

Step 2: Start VULCA

```
vulca serve
```

Opens a local web app in your browser. Full evaluation interface. No cloud account needed. Your data stays on your machine.

Time elapsed: 25 seconds.

> 第二步：启动。vulca serve。在浏览器中打开本地Web应用，完整评估界面，无需云账号，数据留在本地。已用时：25秒。

---

## Tweet 3 — Step 3: Evaluate

Step 3: Upload an AI-generated image, select cultural perspectives, hit Evaluate.

VULCA analyzes across 47 dimensions and 8 cultural frameworks simultaneously.

Time elapsed: 55 seconds.

> 第三步：上传AI生成的图像，选择文化视角，点击评估。VULCA同时跨47个维度和8个文化框架分析。已用时：55秒。

## Tweet 5 — Results

What you get back:

- 6-axis radar chart (Creativity, Technique, Emotion, Context, Innovation, Impact)
- Per-culture breakdown across 8 perspectives
- Specific failure flags (e.g., "Anachronistic cultural elements detected")
- Actionable recommendations

> 你得到的：6轴雷达图、8个文化视角的逐项拆解、特定失败标记（如"检测到时代错误的文化元素"）、可执行的建议。

## Tweet 6 — For Developers

Prefer code over UI? Three lines:

```python
from vulca import evaluate
result = evaluate("path/to/image.png")
print(result.dimensions)
```

Integrate cultural evaluation into your CI/CD pipeline. Catch cultural issues before deployment.

#CulturalAI

> 更喜欢代码？三行搞定。把文化评估集成到你的CI/CD流水线中，部署前捕获文化问题。
