## 1. 会议背景

Phase 3 邮件 (2026-01-25) 发送至 jwang80@wm.edu 后，王晋东回复并约定视频会议。王晋东是 CultureVLM/CultureVerse 论文通讯作者（arXiv 2025.01），VLM 文化理解研究先驱，World Top 2% Highly Cited。

---

## 2. 会议流程

### 2.1 VULCA 项目介绍（俞昊睿主讲）

- 介绍了个人背景和 VULCA 平台当前开发状况
- 展示了已完成的三次展览成果
- 介绍了 vulcaart.art 网站及技术架构

**王晋东反应**: 初期不太理解 VULCA 的定位，但保持了耐心倾听。

### 2.2 商业模式讨论（核心议题）

王晋东对商业模式和技术细节更感兴趣，**未主动提及学术定位**。

**俞昊睿阐述**:
- 目前已有三个展览，有主办方提供资金，完成了网站搭建
- 计划寻求更长久、更稳定的商业方案——与公司和机构合作
- 这需要商业案例分析和资金支持
- 当前需求：显卡/算力、人力、经费，可以逐步推进

**王晋东回应**:
- **无法提供**: 人力、经费
- **可提供算力**: 有显卡和算力资源，但需要走学校内部注册流程
- **时间线**: 明年可能有学生专注文化领域研究，但需要等待

### 2.3 技术方向建议（王晋东核心建议）

王晋东提出了明确的技术方向建议：

> **做生成模型 + Agent，而非继续做识别和测试。**

核心观点：
- 科学家/学者一直在做的识别和测试赛道已经很拥挤
- **生成模型 + Agent 架构**会有更多曝光度和商业前景
- 具体方向：使用**开源模型**，安装插件和组件，**生成二维图像**，基于 VULCA 现有作品
- 文化领域的生成 + Agent 在行业内**比较空缺**，值得尝试
- 如果能做出让**用户可以使用**的产品，这是一个很好的方向

### 2.4 学术定位简要讨论

- CultureVLM 不是王晋东的主线研究方向
- 俞昊睿指出：相关工作中，VULCA 在 AI x 艺术/文化交叉领域的深度无人能及，且有更深入的后续成果
- 王晋东未对此深入讨论，重心始终在商业+技术路线

---

## 3. 关键结论

### 王晋东感兴趣的点
1. VULCA 团队**既有科研成果，又有能力将其商业化**
2. **生成 + Culture** 方向在行业内空缺，有机会
3. **Agent 架构设计 + 开源模型本地部署 + 生成**这条路线有前景

### 合作前提条件
王晋东明确提出了**两步走**方案：

| 步骤 | 内容 | 交付物 |
|------|------|--------|
| **Step 1** | 把 Agent 生成模型的 pipeline 跑通 | 可演示的 working prototype |
| **Step 2** | 提交可上线版本的项目预算报价 | 预算方案文档 |

完成以上两步后，双方再讨论下一步合作。

## 5. 战略评估

**机会**: 王晋东认可 VULCA 在文化 x AI 交叉领域的独特定位，并给出了清晰的技术路线建议（Agent + 生成）。这是从学术评测向**可商业化产品**转型的明确信号。

**挑战**: 短期内无法获得人力和经费支持；算力资源需等学校流程和学生到位。

**下一步优先级**: 先独立完成 Agent + 生成 pipeline 的 prototype，这既是给王晋东的交付物，也是 VULCA 产品化的关键技术验证。
